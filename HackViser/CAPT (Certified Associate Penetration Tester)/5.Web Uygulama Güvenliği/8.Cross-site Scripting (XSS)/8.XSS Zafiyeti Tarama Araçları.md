XSS zafiyetlerini otomatize bir biçimde araçlar ile tespit etmek ya da istismar etmek de mümkündür. Bunun için ücretsiz ve ücretli birçok araç bulunmaktadır. Bunların bir kısmı da açık kaynak olarak paylaşılmıştır. Sık kullanılanlardan bazıları aşağıda verilmiştir.

### XSStrike

XSStrike, gelişmiş bir XSS tespit aracıdır ve Python ile geliştirilmiştir. Fuzzing, DOM tabanlı XSS tespiti ve çeşitli filtrelerden/engellerden kaçınma gibi özelliklere sahiptir. XSStrike, bir URL'yi tarayabilir, parametreleri test edebilir ve XSS saldırı vektörlerini kullanarak Reflected ve Stored XSS zafiyetleri için otomatik testler gerçekleştirebilir. Ayrıca, açıkların sömürülebilirliğini doğrulamak için payloadlar üretebilir.

GitHub Adresi: [https://github.com/s0md3v/XSStrike](https://github.com/s0md3v/XSStrike)

![](https://storage.hackviser.com/file/hackviser-prod/trainings/sections/images/af7e82d5-4e06-4e18-8a1e-d7b589feb1e0/xsstrike-4ca3c94fb.webp)

Kurulum:

XSStrike, GitHub üzerinden indirilebilir. Python3 ve pip ile kurulum yapılabilir:

```auto
git clone https://github.com/s0md3v/XSStrike.git
cd XSStrike
pip3 install -r requirements.txt
```

Kullanım:

XSStrike'ı çalıştırmak için terminalde aşağıdaki gibi bir komut kullanılmalıdır:

```auto
python3 xsstrike.py -u "http://example.com"
```

Bu, belirtilen URL'deki XSS zafiyetlerini tespit etmek için çeşitli testler yapar.

### XSSCon

XSSCon, basit ve hafif bir XSS tarama aracıdır. Python ile geliştirilmiştir ve komut satırı üzerinden çalıştırılır. Bu araç, belirli bir URL'deki XSS zafiyetlerini test etmek için kullanılır. Kullanıcı tarafından belirlenen parametreler üzerinden gönderilen isteklerde XSS açıklarını saptayarak, potansiyel güvenlik risklerini ortaya çıkarır.

GitHub Adresi: [https://github.com/menkrep1337/XSSCon](https://github.com/menkrep1337/XSSCon)

![](https://storage.hackviser.com/file/hackviser-prod/trainings/sections/images/af7e82d5-4e06-4e18-8a1e-d7b589feb1e0/xsscon-f2a8b9078.webp)

Kurulum:

XSSCon, GitHub üzerinden indirilip kullanılabilir. Python ortamının kurulu olduğundan emin olunmalıdır (Python 3 önerilir).

**Gereksinimler:**

```auto
pip install bs4
```

```auto
pip install requests
```

**İndirme:**

```auto
git clone https://github.com/menkrep1337/XSSCon.git
chmod 755 -R XSSCon
cd XSSCon
python3 xsscon.py --help 
```

Kullanım

XSSCon'u çalıştırmak için komut satırına gidilerek ve hedef URL ile birlikte aşağıdaki komut kullanılmalıdır.

```auto
python3 xsscon.py -u "http://example.com"
```

Bu komut, belirtilen URL'deki girdi alanlarını tarar ve XSS zafiyeti için testler gerçekleştirir.

### BeEF (Browser Exploitation Framework)

BeEF, web tarayıcıları üzerinde çalışan bir güvenlik aracıdır ve genellikle XSS zafiyetlerini sömürmek için kullanılır. BeEF, bir kullanıcının tarayıcısına sızdırılan bir JavaScript kodu aracılığıyla kontrol edilebilir. Bu araç, zafiyetli bir site üzerinden kullanıcıların tarayıcılarını ele geçirme ve bu tarayıcılar üzerinden çeşitli güvenlik testleri yapma imkanı tanır.

GitHub Adresi: [https://github.com/beefproject/beef](https://github.com/beefproject/beef)

![](https://storage.hackviser.com/file/hackviser-prod/trainings/sections/images/af7e82d5-4e06-4e18-8a1e-d7b589feb1e0/beef-038be95e2.webp)

Kurulum:

BeEF, Ruby ile yazılmıştır. Öncelikle Ruby ortamının kurulu olduğundan emin olunmalıdır. Daha sonra aşağıda bulunan komutlar ile kurulum işlemi gerçekleştirilmelidir.

```auto
git clone https://github.com/beefproject/beef.git
cd beef
./install
```

Kullanım:

BeEF'i çalıştırdıktan sonra, web arayüzüne [http://localhost:3000/ui/panel](http://localhost:3000/ui/panel) adresinden erişebilir ve yönetici girişi yapabilir(varsayılan parola 'beef'). Buradan, çeşitli modüller kullanılarak tarayıcılarda XSS saldırıları gerçekleştirebilir.
