SQL'de kullanılan özel karakterler, belirli işlevleri yerine getirmek veya veritabanı sorgularında özel anlamlar taşımak için kullanılır. Aşağıdaki tablo, SQL'de sıklıkla kullanılan bazı önemli özel karakterler ve açıklamalarını içermektedir.

|Karakter|Açıklama|Örnek Kullanım|
|---|---|---|
|*|Tüm sütunları seçmek için kullanılır|SELECT * FROM users;|
|%|'LIKE' sorgularında herhangi bir karakter dizisini temsil eder|SELECT * FROM users WHERE name LIKE 'A%';|
|_|'LIKE' sorgularında tek bir karakteri temsil eder|SELECT * FROM users WHERE name LIKE 'A_';|
|'|Karakter dizilerini sınırlamak için kullanılır|SELECT * FROM users WHERE name = 'Jack';|
|"|Sütun ve tablo adlarını sınırlamak için kullanılır|SELECT "name" FROM "users";|
|;|Sorgu sonlandırıcı olarak kullanılır, SQL komutlarını birbirinden ayırır|SELECT * FROM users; SELECT * FROM products;|
|--|Satır sonuna kadar olan kısmı yorum olarak işaretler|SELECT * FROM users -- Bu sorgu tüm kullanıcıları listeler|
|/* */|Çok satırlı yorum blokları için kullanılır|/* Bu yorum birden fazla satırı kapsar */ SELECT * FROM users;|
|+|Aritmetik toplama operatörü, ayrıca stringleri birleştirmek için de kullanılabilir (bazı SQL versiyonlarında)|SELECT name, surname, name + ' ' + surname AS full_name FROM users;|
|,|Sütunları ayırmak için kullanılır|SELECT name, surname, age FROM users;|
|#|MySQL'de tek satırlık yorum başlatıcısı olarak kullanılır|SELECT * FROM users # Bu yorum satırı|
|=|Eşitlik operatörü, iki değerin eşit olup olmadığını kontrol eder|SELECT * FROM users WHERE age = 30;|
|(, )|Fonksiyonlar ve sorgu gruplamalarında kullanılır|SELECT SUM(age) FROM users WHERE age > 20;|
|%00|Null byte, bir karakter dizisinin sonunu belirten özel bir karakterdir ve bazı sistemlerde bir string sonlandırıcı olarak işlev görür. SQL bağlamında, null byte karakteri, bazı durumlarda güvenlik açıklarını keşfetmek veya istismar etmek için kullanılabilir.|SELECT * FROM users WHERE username = 'admin' or char(0) AND password = 'guessme'; # Bu sorguda, char(0) kullanımı teorik olarak null byte karakterini temsil eder ve bu, sorgunun geri kalan kısmını etkisiz hale getirebilir.|
